# DaFiles
A simple file explorer built using Avalonia UI.

<img height="400" alt="DaFiles UI" src="https://github.com/user-attachments/assets/5abe6413-0e29-4b10-8ec8-6f08a5385432" />

## Features
- Local and WebDAV file browsing
- Local directory operations: Cut, copy, paste, delete (permanent and to recycle bin)
- Dual-pane view
